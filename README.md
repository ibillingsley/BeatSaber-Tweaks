# Iza's Beat Saber Tweaks

A collection of tweaks combined into one mod.

All features are optional and disabled by default, enable in ⚙️ Options → Mod Settings

## Tweaks

#### Game Visuals

-   Disable player platform border
-   Disable wall outline bloom

#### Game HUD

-   Adjust font size of score percent
-   Adjust font size of rank

#### Bug Fixes

-   Fix cut score text bug in BS 1.40.0 – 1.40.8 (https://github.com/Meivyn/BeatSaberBugs/issues/26)
-   Clear BetterSongList filter when accepting a BeatSaberPlus song request
